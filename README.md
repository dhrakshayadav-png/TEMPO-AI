# Temperature Prediction using Machine Learning

## Project Overview
This project predicts temperature using Machine Learning based on weather-related features.

## Features
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Decision Tree Regressor Model
- Model Evaluation (MAE, MSE, RMSE, R² Score, Accuracy)
- Temperature Prediction

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
